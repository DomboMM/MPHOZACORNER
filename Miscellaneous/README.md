# Mphoza Corner Kotas

A website for Mphoza Corner Kotas, a fast food restaurant specializing in Kotas in Polokwane, South Africa.

## Features

- Responsive design
- Lightbox gallery for menu items
- Contact form with validation
- Interactive Leaflet map
- SEO optimized
- Tabbed menu categories
- Special offer modal
- FAQ accordion
- Search functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Lightbox2
- Leaflet.js
- OpenStreetMap
